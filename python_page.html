Основы Python

Python — популярный язык программирования. Он был создан Гвидо ван Россумом и выпущен в 1991 году. Он используется для:
  • веб-разработка (серверная часть),
  • разработка программного обеспечения,
  • математика,
  • системные сценарии.

Ввод/вывод

Для ввода данных в Python используется команда input() она позволяет принимать любые вводимые данные, можно сделать ввод только чисел используя int(input()) или же чисел с запятой, используя float(input()), или же только строчные значения string(input())
Для вывода используется команда print(). Чтобы вывести сообщение нужно прописать print(‘Ваше сообщение’), можно выводить также переменную, введя просто название переменной в команду, на примере:
message = “Привет, мир”
print(message)

Оператор условия

Сначала записывается часть if с условным выражением, далее могут следовать одна или более необязательных частей elif, и, наконец, необязательная часть else. Общая форма записи условной инструкции if выглядит следующим образом:
if test1:
 state1
elif test2:
 state2
else:
 state3

Цикл for

Цикл for, также называемый циклом с параметром, в языке Питон богат возможностями. В цикле for указывается переменная и множество значений, по которому будет пробегать переменная. Множество значений может быть задано списком, кортежем, строкой или диапазоном.
i = 1
for color in 'red', 'orange', 'yellow', 'green', 'cyan', 'blue', 'violet':
 print('#', i, ' color of rainbow is ', color, sep = '')
 i += 1
